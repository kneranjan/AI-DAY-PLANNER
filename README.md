# AI Day Planner

## What it does
This is an AI Day Planner which prioritises your taskes use AI. This can be used by anyone who wants help in 
managing their day.
## Tech Stack
- Python
- SQLite
- Gemini API

## How to run it
1. Clone the repo
2. pip install -r requirements.txt
3. Create a .env file based on .env.example
4. Add your Gemini API key
5. python3 main.py
